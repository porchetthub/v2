# 🤝 Come contribuire a PorkettApp

Grazie per il tuo interesse nel contribuire!

## 📝 Aggiungere un nuovo venditore

1. Modifica `src/data/sellers.js`
2. Aggiungi un nuovo oggetto all'array `sellers`
3. Segui la struttura esistente
4. Includi: coordinate GPS, contatti, menu, recensioni

## 🗺️ Aggiungere dati reali

- Google Maps: aggiungi Place ID
- Yelp: aggiungi Business ID  
- Instagram: aggiungi handle ufficiale
- Facebook: aggiungi pagina ufficiale

## 📋 Aggiornare i bandi

1. Modifica `src/data/sellers.js` → array `publicTenders`
2. Aggiungi nuovi bandi comunali
3. Aggiorna lo stato (aperto/chiuso)

## 🐛 Segnalare bug

Apri una [Issue](https://github.com/tuo-username/porkettapp/issues) con:
- Descrizione del problema
- Browser e versione
- Screenshot se utile

## 🚀 Deploy

Ogni push su `main` triggera il deploy automatico su Vercel.
